Chạy streamlit trong terminal
Link dataset: https://www.kaggle.com/datasets/uciml/default-of-credit-card-clients-dataset/data
Cách chạy:
- chạy file main.py: tạo ra file: processed_credit_data.csv. Nếu file nằm ngoài folder chính thì kéo vào
- Chạy file train.py: tạo ra file: multioutput_model.pkl. Nếu file nằm ngoài folder chính cũng kéo vào
- Mở terminal chạy lệnh: streamlit run risk_management\app.py
- file test.csv để test chương trình
